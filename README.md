# Счётчик (ev_Counter)

Минимальное мобильное приложение-счётчик на **React Native** и **Expo**.

## Возможности

- Крупное число по центру экрана (начальное значение — 0)
- Кнопка **+** — увеличивает значение на 1
- Кнопка **−** — уменьшает значение на 1
- Кнопка **Сброс** — сбрасывает значение в 0

## Запуск

1. Установите зависимости:

```powershell
npm install
```

2. Запустите проект:

```powershell
npx expo start
```

3. Откройте приложение:
   - На телефоне: установите **Expo Go** и отсканируйте QR-код из терминала
   - В эмуляторе: нажмите `a` (Android) или `i` (iOS) в терминале после запуска

## Требования

- Node.js 18+
- npm или yarn
